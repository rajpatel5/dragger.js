Draggable

Click on the royalblue coloured box and drag it around. Click on the bottom-right corner of the box and drag up, down, left, or right, this will resize the box. You will notice once you let go, the box immediately seizes to move or resize.



  • JS
  • HTML
  • CSS 
                            
    const draggable = new Draggable('.draggable', true);
    draggable.addListeners();
                            
                        
                            
    <div class="draggable"></div>
                            
                        
                            
    .draggable {
        width: 160px;
        height: 160px;
        position: absolute;
        background-color: royalblue;
        border-radius: 16px;
    }

Droppable

Click on the limegreen coloured box inside the middle container and drag it to any other container. You will notice the box cannot be dropped outside the containers. Once you let go, the box is immediately dropped to the container you were on top of.



  • JS
  • HTML
  • CSS 
                            
    const droppable = new Droppable('.droppable', '.droppable-container');
    droppable.addListeners();
                            
                        
                            
    <div class="droppable-container"></div>
    <div class="droppable-container"></div>
    <div class="droppable-container">
        <div class="droppable"></div>
    </div>
    <div class="droppable-container"></div>
    <div class="droppable-container"></div>
                            
                        
                            
    .droppable {
        position: relative;
        width: 140px;
        height: 140px;
        margin: 6px 0 0 6px;
        border-radius: 16px;
        background-color: limegreen;
        cursor: move;
    }
    
    .droppable-container {
        display: inline-block;
        width: 160px;
        height: 160px;
        margin: 10px;
        border: 4px limegreen solid;
        border-radius: 16px;
    }

Sortable

Click on any of the gold coloured item boxes inside the containers and drag up or down within a container, or to any other container. You will notice the box cannot be dropped outside the containers. Also as you hover over any of the item boxes, a shadow of the item box you are dragging moves in order where ever you are hovering. Some of the boxes cannot be moved since they are declared unsortable. Once you let go, the box is immediately dropped to the position you were on top of.

Rotatable

Click inside the coral bordered container and drag left or right. you will notice that the car spins around, as soon as the mouse moves outside the box it will stop.


Rotate me by dragging left-to-right or right-to-left


  • JS
  • HTML
  • CSS 
                            
    // Uses jQuery
    $(document).ready(function () {
        $('.rotatable').rotatable({
            imgSequencePath: "./images/",
            numOfImgs: 37,
            imgExt: "png",
        });
    });
                            
                        
                            
    <div class="rotatable-container">
        <div class="rotatable"></div>
    </div>                        
                            
                        
                            
    .rotatable-container {
        display: block;
        position: relative;
        margin: 0px auto 0px auto;
        border: 4px coral solid;
        border-radius: 16px;
    }
