GETTING STARTED

To use the Dragger API, You will need jQuery. Add the two following scripts to your html file. You will need the path to the dragger.js library on your local machine. 


                            
    <script src="https://code.jquery.com/jquery-1.11.2.min.js"></script>
    <script src="/path/to/dragger.js"></script>

Draggable

Allows DOM elements to be dragged up, down, left, or right. Also allows for them to be resized to be made bigger or smaller.

Function type Parameters Code Description
Constructor 1.) objectClassName: String

2.) resizablility: Boolean (optional) 
                                     
Draggable(objectClassName, [ resizablility ] );
Creates an instance of the draggable class
Listener None 
                                     
addListeners();
Adds listeners to an instance to allow for interactions


Droppable

Allows DOM elements to be dragged and dropped into specified containers.

Function type Parameters Code Description
Constructor 1.) objectClassName: String

2.) containerClassName: String 
                                     
Droppable(objectClassName, containerClassName);
Creates an instance of the droppable class
Listener None 
                                     
addListeners();
Adds listeners to an instance to allow for interactions


Sortable

Allows multiple DOM elements to be dragged and dropped into specified containers. They can be dropped into any order.

Function type Parameters Code Description
Constructor 1.) objectsClassName: String

2.) containersClassName: String

3.) indicies: Array<Int> (optional) 
                                     
Sortable(objectsClassName, containersClassName, [ indicies ] );
Creates an instance of the sortable class
Listener None 
                                     
addListeners();
Adds listeners to an instance to allow for interactions

Rotatable

Allows multiple DOM elements to be dragged and rotated to create a 360 degree spin effect. They can be dragged left-to-right or right-to-left.

Function type Parameters Code Description
Constructor 1.) objectClassName: String

2.) imagesPath: String

3.) totalImages: Integer

4.) imageExtension: String (Default: png) 
                                     
$(document).ready(function () {
    $(objectClassName).rotatable({
        imgSequencePath: imagesPath,
        numOfImgs: totalImages,
        imgExt: imageExtension,
    });
});
Creates an instance of the rotatable class
